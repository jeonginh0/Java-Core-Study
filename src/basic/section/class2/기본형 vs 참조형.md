# 기본형 vs 참조형
**자바에서 참조형을 제대로 이해하는 것은 중요함**

기본형(Primitive Type): `int`, `double`, `long`, `boolean`등 변수에 사용할 값을 직접 넣을 수 있는 데이터 타입

참조형(Reference Type): `Student student1`, `int[] students`와 같이 데이터에 접근하기 위한 참조(주소)를 저장하는 데이터 타입

## 기본
- 기본형은 숫자 10, 20과 같이 실제 사용하는 값을 변수에 담을 수 있다. 해당 값 바로 사용가능
- 참조형은 실제 사용하는 값을 변수에 담는 것이 아니라, 객체의 위치(참조, 주소)를 저장한다.

## 계산
- 기본형은 들어있는 값을 그대로 계산에 사용할 수 있다.
  - 예) +, -, 사용 등등
- 참조형은 들어있는 참조값을 그대로 사용할 수 없다.
  - 예) 더하고 빼고 사용 못함. 참조값만 가지고는 계산할 수 있는 것이 없음.
```java
// 가능
int a = 10, b = 20;
int sum = a + b;
```

```java
// 불가능
Student s1 = new Student();
Student s2 = new Student();
s1 + s2; // 오류 발생
```

```java
// 가능
Student s1 = new Student();
s1.grade = 100;

Student s2 = new Student();
s2.grade = 90;

int sum = s1.grade + s2.grade; // 연산 가능
```

### TIP !
기본형을 제외한 나머지는 모두 참조형
- 기본형은 소문자로 시작
- 클래스는 대문자로 시작 (참조형)
- 개발자는 참조형인 클래스만 직접 정의 가능.
- String은 클래스. 즉, 참조형. 문자는 매우 자주 다루기 때문에 자바에서 편의 기능을 제공함.

---

## 변수 대입
**대원칙**: 자바는 항상 변수의 값을 복사해서 대입한다.

자바에서 변수에 값을 대입하는 것은 변수에 들어 있는 값을 복사해서 대입하는 것.

기본형이면 변수에 들어 있는 실제 사용하는 값을, 참조형이면 변수에 들어 있는 참조값을 복사해서 대입.
